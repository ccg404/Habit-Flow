# Habit-Flow
HabitFlow is a lightweight web application designed to help users build consistency by tracking daily goals. This project was built to master the fundamentals of CRUD (Create, Read, Update, Delete) operations and state management in a web environment.
🚀 Key Features
Create: Add custom habits with unique titles and categories.

Read: A clean dashboard to view all active habits and their current status.

Update: Toggle habits as "Complete" for the day or edit habit details.

Delete: Remove habits you no longer wish to track to keep your dashboard focused.
